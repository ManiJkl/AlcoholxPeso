# Alcohol x Peso

A simple web calculator to help you find the best value when buying alcoholic beverages.

## Features

- Calculate cost per milliliter of beverage
- Calculate pure alcohol volume based on percentage
- Calculate cost per milliliter of pure alcohol
- Compare multiple beverages side-by-side
- Automatically highlights the best value option

## How to Use

1. Open `index.html` in your web browser
2. Enter the price (in Peso), volume (in ml), and alcohol content (%) of a beverage
3. Click "Calculate Value" to see the results
4. Click "Add to Comparison" to add the beverage to your comparison list
5. Add more beverages to compare and find the best value!

## Quick Start

Simply open the `index.html` file in any modern web browser. No installation or server required!

## Files

- `index.html` - Main HTML structure
- `styles.css` - Styling and layout
- `script.js` - Calculator functionality and comparison logic
